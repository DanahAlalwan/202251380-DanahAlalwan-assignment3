# Assignment 3 – Advanced Portfolio Website

## Project Description
This project is an advanced version of my portfolio website built for Assignment 3.  
It extends my previous portfolio by adding:
- GitHub API integration
- advanced project filtering and sorting
- state management using localStorage
- improved contact form validation
- performance improvements and better documentation

## Features
- Responsive portfolio website
- Dark/light theme toggle with saved preference
- Dynamic project filtering, sorting, and search
- Live GitHub repositories fetched from GitHub API
- Personalized greeting stored with localStorage
- Contact form validation
- Clean and organized code structure
