# Assignment 3 – Advanced Portfolio Website

## Project Description
This project is an advanced version of my portfolio website developed for Assignment 3.  
It builds on my previous assignments by adding more advanced functionality, including external API integration, complex JavaScript logic, state management, performance improvements, and project documentation.

The goal of this project is to create a more dynamic and interactive portfolio website while following good coding practices and responsible AI usage.

## Features
- Responsive portfolio website
- Dark/light mode toggle with saved preference using `localStorage`
- Dynamic project filtering by category
- Project sorting by title and date
- Project search functionality
- Live GitHub repositories fetched using the GitHub API
- Visitor name storage using `localStorage`
- Contact form validation with multiple checks
- Clean file organization and documentation

## Technologies Used
- HTML5
- CSS3
- JavaScript (ES6)
- GitHub REST API
