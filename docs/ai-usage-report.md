# AI Usage Report

## Tools Used & Use Cases
I used ChatGPT to:
- understand the requirements of Assignment 3
- plan how to extend my Assignment 2 portfolio
- generate ideas for API integration
- debug JavaScript logic
- improve documentation and code structure

## Benefits & Challenges
AI helped me save time and understand how to implement advanced features such as API fetching, sorting, and localStorage.

One challenge was that AI-generated code needed review and modification to match my own project structure and style. Some suggestions also needed testing before they worked correctly.

## Learning Outcomes
Through using AI, I learned:
- how to fetch live data from an external API
- how to handle asynchronous JavaScript using `fetch()` and `async/await`
- how to combine filtering, sorting, and search logic
- how to improve code organization and documentation

## Responsible Use & Modifications
I reviewed all AI-generated suggestions carefully, tested them, and modified them to ensure correctness and originality.

I did not submit unmodified AI output directly. I used AI as a learning and support tool, then adapted the final solution to fit my assignment requirements and my portfolio design.
