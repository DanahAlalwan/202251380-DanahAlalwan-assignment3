# AI Usage Report

## Tools Used & Use Cases
I used ChatGPT as an AI support tool during this assignment.

It helped me with:
- understanding the Assignment 3 requirements
- planning how to extend my Assignment 2 portfolio into a more advanced version
- generating example code for GitHub API integration
- improving JavaScript logic for filtering, sorting, searching, and validation
- improving documentation such as the README and technical documentation

## Benefits & Challenges
Using AI helped me save time and understand advanced concepts more clearly, especially:
- API integration using `fetch()`
- asynchronous JavaScript with `async/await`
- combining multiple conditions in filtering, sorting, and searching
- improving project structure and written documentation

One challenge was that AI-generated suggestions were not always directly suitable for my project.  
Some code examples needed editing to match my HTML structure, naming, and design choices.  
I also had to test the code manually and fix small issues before using it.

## Learning Outcomes
From using AI during this project, I learned:
- how to fetch live data from an external API
- how to handle request errors using `try...catch`
- how to manage application state with `localStorage`
- how to write more structured JavaScript code
- how to improve documentation and explain my implementation clearly

## Responsible Use & Modifications
I used AI responsibly as a learning and support tool, not as a replacement for my own work.

To ensure correctness and academic integrity, I:
- reviewed all AI-generated suggestions carefully
- tested the code before keeping it
- modified the suggestions to fit my own portfolio structure
- improved and rewrote parts of the code and documentation in my own way
- made sure I understood how each feature works before submitting the project

This project reflects my own understanding, testing, and final modifications.
